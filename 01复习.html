1:响应式原理.(数据更新，视图不更新) vue通过Object.defineProperty(对象,对象属性,{ 4种描述, get(){}, set(){} }); 一个对象属性在设置了get和set后,其value值就只能通过get返回. vue会给data数据的所有属性设置getter和setter,这样就可以监听到data的变化,从而相应的改变视图. 数据变化,视图不更新: 1):只修改数组的元素,而不是修改数组.除非这个元素是引用类型. 能够修改数组本身的方法,push,pop,unshift,shift,splice,reverse,sort. 字符串的所有方法都不修改字符串本身.(如果需要修改,得重新赋值) 2):修改一个本来不存在的对象属性. 当视图更新后，系统会再次检测data内的数据，如果需要添加新的监听，会自动添加。 2:watch 用于在数据变化时做一些额外事情.(监听数据变化,实现一些自定义逻辑). watch可以监听的数据包括data和计算属性 data:{name} watch:{ name(newVal,oldVal){ 自定义逻辑 }, "name":{ deep:true, immediate:true, handler(newVal,oldVal){ } }, "name":[函数1,函数2,函数3], "name":['name',function(){},{ handler(){} }, } 3:computed 如果一个数据是依赖于别的数据计算得出，就需要计算属性。 简写的函数就是全写的get函数. 计算属性无法解决异步操作的问题.watch可以. computed:{ 数据名(){ return 计算属性的值 }, 数据名:{ get(){},set(){} } }
